### 🚫 Deprecated Features (Do Not Migrate)

<Callout type="error">
Meta deprecated these features in 2024. They should NOT be implemented in Go.
</Callout>

#### Facebook Groups API
- **Deprecated**: January 2024 (v19.0)
- **Impact**: Cannot fetch group posts or member analytics
- **Alternative**: None available

#### Page Fans/Audience Insights
- **Deprecated**: March 14, 2024
- **Affected Metrics**:
  - `page_fans_city` - Geographic distribution
  - `page_fans_country` - Country demographics
  - `page_fans_gender_age` - Age and gender breakdown
  - `page_fans_by_like_source` - Attribution (shows "Unknown" only)
  - `page_fans_online` - Active times

## Known Issues & Workarounds

### Facebook Platform

<Tabs items={['Groups API', 'Fans Demographics', 'Token Expiration']}>
  <Tab value="Groups API">
    **Issue**: Groups API deprecated in v19.0
    - **Impact**: Cannot fetch group posts
    - **Workaround**: None - feature sunset by Meta
    - **Resolution**: Remove from product offering
  </Tab>
  
  <Tab value="Fans Demographics">
    **Issue**: Fans demographics removed March 2024
    - **Impact**: No audience insights available
    - **Workaround**: Use Facebook Business Manager UI
    - **Resolution**: Educate customers about API limitations
  </Tab>
  
  <Tab value="Token Expiration">
    **Issue**: Token expiration not handled automatically
    - **Impact**: Manual token refresh required
    - **Workaround**: Monitor token validity, refresh manually
    - **Resolution**: Implement auto-refresh in Phase 1
  </Tab>
</Tabs>

### Instagram Platform

<Tabs items={['Stories Insights', 'Media Insights']}>
  <Tab value="Stories Insights">
    **Issue**: Stories insights not implemented
    - **Impact**: Missing story performance data
    - **Workaround**: Continue using Python service for stories
    - **Resolution**: Implement in Phase 2
  </Tab>
  
  <Tab value="Media Insights">
    **Issue**: Media insights require separate API calls
    - **Impact**: Increased API usage and latency
    - **Workaround**: Batch requests where possible
    - **Resolution**: Optimize batching strategy
  </Tab>
